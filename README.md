## ðŸŽ¯ About the Project
This project is a pixel-perfect static clone of the Cursor AI code editor's homepage. The goal was to replicate the visual design and layout fidelity of a modern SaaS landing page using **pure HTML and CSS**.

**Note:** This project is currently optimized for **Desktop View** (1366px+ screens).

## âœ¨ Key Features
- **Semantic Architecture**: Uses proper HTML5 tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`) rather than generic divs.
- **Modern CSS Layouts**: Extensive use of **CSS Grid** and **Flexbox** to manage complex alignments (e.g., the partner logo grid).
- **Theme Management**: Centralized color control using **CSS Variables** (Custom Properties) for easy theming.
- **Advanced Color Spaces**: Utilization of `oklab()` for high-fidelity color rendering.
- **Custom Typography**: Integration of the `Inter` font family via Google Fonts to match the original branding.

## ðŸ›  Technologies Used
* **HTML5**: Structure and semantics.
* **CSS3**: Styling, layout, and theming.
* **Google Fonts**: 'Inter' font family.
* **SVG**: Inline vector graphics for high-quality logos and icons.
